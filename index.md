    <article class="section-card">
      <h2>Tastes</h2>
      <ul class="highlight-list">
        <li><strong>Local-first, self-hostable, boring-on-purpose.</strong> If it can run on my own box without a third-party dashboard between me and the data, that's the version I want.</li>
        <li><strong>Agents as teammates, not magic.</strong> Sharp tools — Claude Code, MCP servers, agent skills — beat sprawling autonomous setups. I'd rather give an agent a small, well-named knife.</li>
        <li><strong>Open source as default.</strong> Forks over rewrites. Read the source before reading the marketing page.</li>
        <li><strong>Native &gt; Electron.</strong> A Mac app that respects the OS, a CLI that respects my shell, a config that respects my dotfiles.</li>
        <li><strong>Aesthetic: quiet.</strong> Whitespace, one accent color, type that doesn't shout. Restraint is a feature.</li>
      </ul>
    </article>
